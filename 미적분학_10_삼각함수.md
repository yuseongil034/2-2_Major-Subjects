# 삼각함수의 정의와 성질 - 공대 실전 가이드

## 1. 삼각함수 기본 정의

### 1.1 단위원에서의 정의

**단위원**: $$x^2 + y^2 = 1$$

점 $$P(\cos\theta, \sin\theta)$$에 대해:
- $$\sin\theta = y$$ (세로좌표)
- $$\cos\theta = x$$ (가로좌표)
- $$\tan\theta = \frac{y}{x} = \frac{\sin\theta}{\cos\theta}$$ ($$\cos\theta \neq 0$$)

### 1.2 직각삼각형에서의 정의

직각삼각형에서 각 $$\theta$$에 대해:
- $$\sin\theta = \frac{\text{대변}}{\text{빗변}}$$
- $$\cos\theta = \frac{\text{인접변}}{\text{빗변}}$$
- $$\tan\theta = \frac{\text{대변}}{\text{인접변}}$$

### 1.3 역함수들

- $$\csc\theta = \frac{1}{\sin\theta}$$ (코시컨트)
- $$\sec\theta = \frac{1}{\cos\theta}$$ (시컨트)
- $$\cot\theta = \frac{1}{\tan\theta} = \frac{\cos\theta}{\sin\theta}$$ (코탄젠트)

## 2. 특수각의 삼각함수값

### 2.1 기본 특수각 (외우기 필수)

| $$\theta$$ | $$0°$$ | $$30°$$ | $$45°$$ | $$60°$$ | $$90°$$ |
|------------|--------|---------|---------|---------|---------|
| **라디안** | $$0$$ | $$\frac{\pi}{6}$$ | $$\frac{\pi}{4}$$ | $$\frac{\pi}{3}$$ | $$\frac{\pi}{2}$$ |
| $$\sin\theta$$ | $$0$$ | $$\frac{1}{2}$$ | $$\frac{\sqrt{2}}{2}$$ | $$\frac{\sqrt{3}}{2}$$ | $$1$$ |
| $$\cos\theta$$ | $$1$$ | $$\frac{\sqrt{3}}{2}$$ | $$\frac{\sqrt{2}}{2}$$ | $$\frac{1}{2}$$ | $$0$$ |
| $$\tan\theta$$ | $$0$$ | $$\frac{1}{\sqrt{3}}$$ | $$1$$ | $$\sqrt{3}$$ | $$\infty$$ |

### 2.2 암기 팁

**사인값 패턴**: $$\sin\theta = \frac{\sqrt{n}}{2}$$ ($$n = 0, 1, 2, 3, 4$$)
- $$\sin 0° = \frac{\sqrt{0}}{2} = 0$$
- $$\sin 30° = \frac{\sqrt{1}}{2} = \frac{1}{2}$$
- $$\sin 45° = \frac{\sqrt{2}}{2}$$
- $$\sin 60° = \frac{\sqrt{3}}{2}$$
- $$\sin 90° = \frac{\sqrt{4}}{2} = 1$$

**코사인값**: $$\sin$$과 반대 순서

## 3. 삼각함수의 부호와 사분면

### 3.1 사분면별 부호

| 사분면 | $$\sin\theta$$ | $$\cos\theta$$ | $$\tan\theta$$ |
|--------|----------------|----------------|----------------|
| **I** | $$+$$ | $$+$$ | $$+$$ |
| **II** | $$+$$ | $$-$$ | $$-$$ |
| **III** | $$-$$ | $$-$$ | $$+$$ |

### 3.2 주기성

- $$\sin(\theta + 2\pi) = \sin\theta$$
- $$\cos(\theta + 2\pi) = \cos\theta$$
- $$\tan(\theta + \pi) = \tan\theta$$

## 4. 삼각함수 덧셈정리

### 4.1 기본 덧셈정리

**사인의 덧셈정리**:
$$\sin(\alpha + \beta) = \sin\alpha\cos\beta + \cos\alpha\sin\beta$$
$$\sin(\alpha - \beta) = \sin\alpha\cos\beta - \cos\alpha\sin\beta$$

**코사인의 덧셈정리**:
$$\cos(\alpha + \beta) = \cos\alpha\cos\beta - \sin\alpha\sin\beta$$
$$\cos(\alpha - \beta) = \cos\alpha\cos\beta + \sin\alpha\sin\beta$$

**탄젠트의 덧셈정리**:
$$\tan(\alpha + \beta) = \frac{\tan\alpha + \tan\beta}{1 - \tan\alpha\tan\beta}$$
$$\tan(\alpha - \beta) = \frac{\tan\alpha - \tan\beta}{1 + \tan\alpha\tan\beta}$$

### 4.2 배각공식

**2배각 공식**:
$$\sin 2\theta = 2\sin\theta\cos\theta$$
$$\cos 2\theta = \cos^2\theta - \sin^2\theta = 2\cos^2\theta - 1 = 1 - 2\sin^2\theta$$
$$\tan 2\theta = \frac{2\tan\theta}{1 - \tan^2\theta}$$

**3배각 공식**:
$$\sin 3\theta = 3\sin\theta - 4\sin^3\theta$$
$$\cos 3\theta = 4\cos^3\theta - 3\cos\theta$$

### 4.3 반각공식

$$\sin\frac{\theta}{2} = \pm\sqrt{\frac{1 - \cos\theta}{2}}$$
$$\cos\frac{\theta}{2} = \pm\sqrt{\frac{1 + \cos\theta}{2}}$$
$$\tan\frac{\theta}{2} = \pm\sqrt{\frac{1 - \cos\theta}{1 + \cos\theta}} = \frac{\sin\theta}{1 + \cos\theta} = \frac{1 - \cos\theta}{\sin\theta}$$

## 5. 삼각함수 곱셈-덧셈 공식

### 5.1 곱을 합으로

$$\sin A \cos B = \frac{1}{2}[\sin(A+B) + \sin(A-B)]$$
$$\cos A \sin B = \frac{1}{2}[\sin(A+B) - \sin(A-B)]$$
$$\cos A \cos B = \frac{1}{2}[\cos(A+B) + \cos(A-B)]$$
$$\sin A \sin B = \frac{1}{2}[\cos(A-B) - \cos(A+B)]$$

### 5.2 합을 곱으로

$$\sin A + \sin B = 2\sin\frac{A+B}{2}\cos\frac{A-B}{2}$$
$$\sin A - \sin B = 2\cos\frac{A+B}{2}\sin\frac{A-B}{2}$$
$$\cos A + \cos B = 2\cos\frac{A+B}{2}\cos\frac{A-B}{2}$$
$$\cos A - \cos B = -2\sin\frac{A+B}{2}\sin\frac{A-B}{2}$$

## 6. 실전 계산 예시

### 6.1 특수각 응용 계산

**예시 1**: $$\sin 75°$$ 계산

**해법**:

**단계 1**: $$75° = 45° + 30°$$로 분해

**단계 2**: 덧셈정리 적용
$$\sin 75° = \sin(45° + 30°) = \sin 45°\cos 30° + \cos 45°\sin 30°$$

**단계 3**: 특수각값 대입
$$= \frac{\sqrt{2}}{2} \cdot \frac{\sqrt{3}}{2} + \frac{\sqrt{2}}{2} \cdot \frac{1}{2} = \frac{\sqrt{6} + \sqrt{2}}{4}$$

### 6.2 배각공식 응용

**예시 2**: $$\cos^2 15° - \sin^2 15°$$ 계산

**해법**:

**단계 1**: $$\cos 2\theta = \cos^2\theta - \sin^2\theta$$ 인식

**단계 2**: 공식 적용
$$\cos^2 15° - \sin^2 15° = \cos(2 \cdot 15°) = \cos 30° = \frac{\sqrt{3}}{2}$$

### 6.3 반각공식 응용

**예시 3**: $$\tan 22.5°$$ 계산

**해법**:

**단계 1**: $$22.5° = \frac{45°}{2}$$로 인식

**단계 2**: 반각공식 적용
$$\tan 22.5° = \tan\frac{45°}{2} = \frac{1 - \cos 45°}{\sin 45°}$$

**단계 3**: 특수각값 대입
$$= \frac{1 - \frac{\sqrt{2}}{2}}{\frac{\sqrt{2}}{2}} = \frac{2 - \sqrt{2}}{\sqrt{2}} = \frac{(2 - \sqrt{2})\sqrt{2}}{2} = \sqrt{2} - 1$$

## 7. 삼각방정식 풀이

### 7.1 기본 삼각방정식

**유형 1**: $$\sin\theta = a$$ 형태

**해법**:
- $$|a| \leq 1$$인지 확인
- 주해: $$\theta = \arcsin a$$
- 일반해: $$\theta = \arcsin a + 2\pi n$$ 또는 $$\theta = \pi - \arcsin a + 2\pi n$$

**유형 2**: $$\cos\theta = a$$ 형태

**해법**:
- $$|a| \leq 1$$인지 확인
- 주해: $$\theta = \arccos a$$
- 일반해: $$\theta = \pm\arccos a + 2\pi n$$

### 7.2 고차 삼각방정식

**예시 4**: $$2\sin^2\theta - \sin\theta - 1 = 0$$ 풀이

**해법**:

**단계 1**: $$t = \sin\theta$$로 치환
$$2t^2 - t - 1 = 0$$

**단계 2**: 이차방정식 해결
$$(2t + 1)(t - 1) = 0$$
$$t = -\frac{1}{2}$$ 또는 $$t = 1$$

**단계 3**: 원래 변수로 환원
- $$\sin\theta = -\frac{1}{2}$$ → $$\theta = \frac{7\pi}{6} + 2\pi n, \frac{11\pi}{6} + 2\pi n$$
- $$\sin\theta = 1$$ → $$\theta = \frac{\pi}{2} + 2\pi n$$

## 8. 삼각함수 적분과 미분

### 8.1 기본 미분공식

$$\frac{d}{dx}\sin x = \cos x$$
$$\frac{d}{dx}\cos x = -\sin x$$
$$\frac{d}{dx}\tan x = \sec^2 x$$
$$\frac{d}{dx}\sec x = \sec x \tan x$$
$$\frac{d}{dx}\csc x = -\csc x \cot x$$
$$\frac{d}{dx}\cot x = -\csc^2 x$$

### 8.2 기본 적분공식

$$\int \sin x \, dx = -\cos x + C$$
$$\int \cos x \, dx = \sin x + C$$
$$\int \tan x \, dx = -\ln|\cos x| + C = \ln|\sec x| + C$$
$$\int \sec x \, dx = \ln|\sec x + \tan x| + C$$
$$\int \csc x \, dx = -\ln|\csc x + \cot x| + C$$
$$\int \cot x \, dx = \ln|\sin x| + C$$

### 8.3 삼각함수 적분 기법

**거듭제곱 적분**:
$$\int \sin^n x \cos^m x \, dx$$

**경우 1**: $$m$$이 홀수일 때
$$\cos x$$를 하나 빼고 나머지를 $$\sin x$$로 표현

**경우 2**: $$n$$이 홀수일 때  
$$\sin x$$를 하나 빼고 나머지를 $$\cos x$$로 표현

**경우 3**: 둘 다 짝수일 때
반각공식 사용

## 9. 실수하기 쉬운 포인트

### 9.1 각도 단위 실수

**실수 1**: 도와 라디안 혼동
- **해결책**: 계산기 모드 확인 (DEG/RAD)

### 9.2 부호 실수

- **해결책**: 단위원 그려서 확인
- **팁**: "ASTC" 암기법 활용

- $$\arcsin x \in [-\frac{\pi}{2}, \frac{\pi}{2}]$$
- $$\arccos x \in [0, \pi]$$
- $$\arctan x \in (-\frac{\pi}{2}, \frac{\pi}{2})$$

### 9.3 공식 적용 실수

**실수 5**: 덧셈정리에서 부호 실수
- $$\cos(\alpha - \beta) = \cos\alpha\cos\beta + \sin\alpha\sin\beta$$ ($$+$$)
- $$\cos(\alpha + \beta) = \cos\alpha\cos\beta - \sin\alpha\sin\beta$$ ($${-}$$)

**실수 6**: 반각공식의 부호 결정
- **해결책**: 각이 속한 사분면으로 부호 결정

## 10. 고급 테크닉

### 10.1 삼각치환

**적분에서의 활용**:
- $$\sqrt{a^2 - x^2}$$ → $$x = a\sin\theta$$
- $$\sqrt{a^2 + x^2}$$ → $$x = a\tan\theta$$  
- $$\sqrt{x^2 - a^2}$$ → $$x = a\sec\theta$$

### 10.2 복소수와 삼각함수

**오일러 공식**:
$$e^{i\theta} = \cos\theta + i\sin\theta$$

**드 무아브르 정리**:
$$(\cos\theta + i\sin\theta)^n = \cos n\theta + i\sin n\theta$$

### 10.3 삼각함수의 테일러 급수

$$\sin x = \sum_{n=0}^{\infty} \frac{(-1)^n x^{2n+1}}{(2n+1)!} = x - \frac{x^3}{3!} + \frac{x^5}{5!} - \cdots$$

$$\cos x = \sum_{n=0}^{\infty} \frac{(-1)^n x^{2n}}{(2n)!} = 1 - \frac{x^2}{2!} + \frac{x^4}{4!} - \cdots$$

## 11. 물리학과 공학에서의 응용

### 11.1 파동 방정식

**일반 파동**:
$$y(x,t) = A\sin(kx - \omega t + \phi)$$

여기서:
- $$A$$: 진폭
- $$k$$: 파수 ($$k = \frac{2\pi}{\lambda}$$)
- $$\omega$$: 각진동수
- $$\phi$$: 위상차

### 11.2 교류 회로

**교류 전압**:
$$V(t) = V_0 \sin(\omega t + \phi)$$

**임피던스**:
$$Z = R + jX = |Z|e^{j\phi}$$

여기서 $$\tan\phi = \frac{X}{R}$$

### 11.3 진동과 파동

**단순조화운동**:
$$x(t) = A\cos(\omega t + \phi)$$
$$v(t) = -A\omega\sin(\omega t + \phi)$$
$$a(t) = -A\omega^2\cos(\omega t + \phi)$$

## 12. 계산 최적화 팁

### 12.1 빠른 계산법

**특수각 조합**:
- $$15° = 45° - 30°$$
- $$75° = 45° + 30°$$
- $$105° = 60° + 45°$$
- $$165° = 180° - 15°$$

**대칭성 활용**:
- $$\sin(180° - \theta) = \sin\theta$$
- $$\cos(180° - \theta) = -\cos\theta$$
- $$\sin(90° - \theta) = \cos\theta$$

### 12.2 근사공식 (작은 각에 대해)

$$\theta$$가 라디안으로 작을 때:
$$\sin\theta \approx \theta$$
$$\cos\theta \approx 1 - \frac{\theta^2}{2}$$
$$\tan\theta \approx \theta$$

### 12.3 검산 방법

**피타고라스 정리**:
$$\sin^2\theta + \cos^2\theta = 1$$

**범위 확인**:
- $$-1 \leq \sin\theta \leq 1$$
- $$-1 \leq \cos\theta \leq 1$$
- $$\tan\theta \in \mathbb{R}$$ (정의역에서)
