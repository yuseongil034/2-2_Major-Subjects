# 복소수와 오일러 공식 - 공대 실전 가이드

## 1. 복소수 기본 연산

### 1.1 복소수 표현법

**직교형 (Rectangular Form)**
$$z = a + bi$$

**극형 (Polar Form)**  
$$z = r(\cos\theta + i\sin\theta) = re^{i\theta}$$

**변환 공식**
- $$r = |z| = \sqrt{a^2 + b^2}$$
- $$\theta = \arg(z) = \arctan\left(\frac{b}{a}\right)$$ (사분면 주의)
- $$a = r\cos\theta$$, $$b = r\sin\theta$$

### 1.2 사분면별 각도 계산

**사분면 판정 및 각도 보정**
- **1사분면**: $$a > 0, b > 0$$ → $$\theta = \arctan(b/a)$$
- **2사분면**: $$a < 0, b > 0$$ → $$\theta = \pi + \arctan(b/a)$$
- **3사분면**: $$a < 0, b < 0$$ → $$\theta = \pi + \arctan(b/a)$$
- **4사분면**: $$a > 0, b < 0$$ → $$\theta = 2\pi + \arctan(b/a)$$

### 1.3 기본 연산

**덧셈과 뺄셈** (직교형으로)
$$(a + bi) \pm (c + di) = (a \pm c) + (b \pm d)i$$

**곱셈**
- **직교형**: $$(a + bi)(c + di) = (ac - bd) + (ad + bc)i$$
- **극형**: $$r_1e^{i\theta_1} \cdot r_2e^{i\theta_2} = r_1r_2e^{i(\theta_1+\theta_2)}$$

**나눗셈**
- **직교형**: $$\frac{a + bi}{c + di} = \frac{(a + bi)(c - di)}{c^2 + d^2} = \frac{ac + bd}{c^2 + d^2} + \frac{bc - ad}{c^2 + d^2}i$$
- **극형**: $$\frac{r_1e^{i\theta_1}}{r_2e^{i\theta_2}} = \frac{r_1}{r_2}e^{i(\theta_1-\theta_2)}$$

## 2. 오일러 공식과 응용

### 2.1 오일러 공식

**기본 공식**
$$e^{i\theta} = \cos\theta + i\sin\theta$$

**특수한 경우들**
- $$e^{i\pi} = -1$$ (오일러 등식)
- $$e^{i\pi/2} = i$$
- $$e^{i\pi/4} = \frac{1}{\sqrt{2}} + \frac{i}{\sqrt{2}}$$
- $$e^{-i\theta} = \cos\theta - i\sin\theta$$

### 2.2 삼각함수 표현

**지수형으로 표현**
$$\cos\theta = \frac{e^{i\theta} + e^{-i\theta}}{2}$$
$$\sin\theta = \frac{e^{i\theta} - e^{-i\theta}}{2i}$$

**쌍곡함수와의 관계**
$$\cosh(i\theta) = \cos\theta$$
$$\sinh(i\theta) = i\sin\theta$$

## 3. 복소수 거듭제곱과 드 무아브르 정리

### 3.1 드 무아브르 정리

**공식**
$$[r(\cos\theta + i\sin\theta)]^n = r^n(\cos n\theta + i\sin n\theta)$$

**지수형으로**
$$(re^{i\theta})^n = r^ne^{in\theta}$$

### 3.2 거듭제곱 계산 예시

**예시 1**: $$(1 + i)^8$$ 계산

**해법**:
```
단계 1: 극형으로 변환
1 + i = √2 · e^(iπ/4)

단계 2: 거듭제곱 적용
(1 + i)^8 = (√2)^8 · e^(i·8π/4) = 16 · e^(2πi) = 16 · 1 = 16
```

**예시 2**: $$(\frac{1}{2} + \frac{\sqrt{3}}{2}i)^{12}$$ 계산

**해법**:
```
단계 1: 극형으로 변환
r = √((1/2)² + (√3/2)²) = 1
θ = arctan(√3) = π/3
z = e^(iπ/3)

단계 2: 거듭제곱 적용
z^12 = e^(i·12π/3) = e^(4πi) = 1
```

## 4. 복소수 n제곱근

### 4.1 n제곱근 공식

복소수 $$z = re^{i\theta}$$의 $$n$$제곱근:

$$w_k = \sqrt[n]{r} \cdot e^{i(\theta + 2\pi k)/n}$$

여기서 $$k = 0, 1, 2, ..., n-1$$

### 4.2 단위근 (Roots of Unity)

**n차 단위근**: $$z^n = 1$$의 해

$$\omega_k = e^{2\pi i k/n}, \quad k = 0, 1, ..., n-1$$

**중요한 성질**:
- $$\omega_0 = 1$$ (주단위근)
- $$\omega_k^n = 1$$
- $$\sum_{k=0}^{n-1} \omega_k^j = 0$$ (j ≠ 0일 때)

### 4.3 제곱근 계산 예시

**예시 3**: $$\sqrt{-8i}$$ 계산

**해법**:
```
단계 1: 극형으로 변환
-8i = 8e^(-iπ/2)

단계 2: 제곱근 공식 적용
w₀ = √8 · e^(-iπ/4) = 2√2 · e^(-iπ/4) = 2√2(cos(-π/4) + i sin(-π/4))
   = 2√2(1/√2 - i/√2) = 2 - 2i

w₁ = √8 · e^(i(-π/2 + 2π)/2) = 2√2 · e^(i3π/4)
   = 2√2(-1/√2 + i/√2) = -2 + 2i
```

## 5. 복소함수와 지수함수

### 5.1 복소지수함수

**일반형**
$$e^{z} = e^{a+bi} = e^a(\cos b + i\sin b)$$

**중요한 성질**
- $$|e^{z}| = e^{\text{Re}(z)} = e^a$$
- $$\arg(e^z) = \text{Im}(z) = b$$ ($$\mod 2\pi$$)
- $$e^{z_1} \cdot e^{z_2} = e^{z_1 + z_2}$$

### 5.2 복소로그함수

**주값 (Principal Value)**
$$\log z = \ln|z| + i\arg(z)$$

**일반해**
$$\log z = \ln|z| + i(\arg(z) + 2\pi k), \quad k \in \mathbb{Z}$$

### 5.3 복소거듭제곱

**일반 거듭제곱**
$$z^w = e^{w \log z}$$

**예시 4**: $$i^i$$ 계산

**해법**:
```
단계 1: 로그 계산
log i = ln|i| + i arg(i) = 0 + i(π/2) = iπ/2

단계 2: 거듭제곱 계산
i^i = e^(i · log i) = e^(i · iπ/2) = e^(-π/2) ≈ 0.2079
```

## 6. 삼각함수와 쌍곡함수의 복소확장

### 6.1 복소삼각함수

**정의**
$$\cos z = \frac{e^{iz} + e^{-iz}}{2}$$
$$\sin z = \frac{e^{iz} - e^{-iz}}{2i}$$

**실부와 허부 분리**
$$\cos(x + iy) = \cos x \cosh y - i\sin x \sinh y$$
$$\sin(x + iy) = \sin x \cosh y + i\cos x \sinh y$$

### 6.2 복소쌍곡함수

**정의**
$$\cosh z = \frac{e^z + e^{-z}}{2}$$
$$\sinh z = \frac{e^z - e^{-z}}{2}$$

**삼각함수와의 관계**
$$\cos(iz) = \cosh z$$
$$\sin(iz) = i\sinh z$$

## 7. 실전 문제 유형별 해법

### 7.1 복소방정식 풀이

**유형 1**: $$z^n = a + bi$$ 형태

**해법 단계**:
```
1단계: a + bi를 극형으로 변환
2단계: n제곱근 공식 적용
3단계: 모든 근 나열
```

**예시 5**: $$z^3 = -8$$ 풀이

**해법**:
```
단계 1: -8 = 8e^(iπ)

단계 2: 3제곱근 계산
z₀ = 2e^(iπ/3) = 2(1/2 + i√3/2) = 1 + i√3
z₁ = 2e^(i(π + 2π)/3) = 2e^(iπ) = -2
z₂ = 2e^(i(π + 4π)/3) = 2e^(i5π/3) = 2(1/2 - i√3/2) = 1 - i√3
```

### 7.2 삼각함수 계산

**유형 2**: $$\cos^n\theta$$, $$\sin^n\theta$$ 계산

**체비셰프 다항식 이용**
$$\cos^n\theta = \frac{1}{2^{n-1}}\sum_{k=0}^{\lfloor n/2 \rfloor} \binom{n}{k} \cos((n-2k)\theta)$$

**이항정리 이용**
$$(\cos\theta + i\sin\theta)^n = \sum_{k=0}^{n} \binom{n}{k} \cos^{n-k}\theta (i\sin\theta)^k$$

### 7.3 적분 계산

**유형 3**: 복소함수 적분

**잔류정리 (Residue Theorem)** 응용
$$\oint_C f(z) dz = 2\pi i \sum \text{Res}(f, z_k)$$

## 8. 실수하기 쉬운 포인트

### 8.1 각도 계산 실수

**실수 1**: 사분면 판정 오류
- **해결책**: 실부와 허부의 부호를 모두 확인
- **체크 방법**: 계산기의 atan2 함수와 비교

**실수 2**: 각도의 주기성 무시
- **주의점**: $$e^{i\theta} = e^{i(\theta + 2\pi k)}$$
- **해결책**: 필요시 $$[0, 2\pi)$$ 구간으로 정규화

### 8.2 거듭제곱 계산 실수

**실수 3**: 드 무아브르 정리 적용 시 각도 실수
- **주의점**: $$n\theta$$가 $$2\pi$$의 배수인지 확인
- **팁**: 계산 전에 $$\theta$$를 정확히 구하기

**실수 4**: n제곱근에서 모든 근을 구하지 않음
- **해결책**: $$k = 0, 1, 2, ..., n-1$$ 모두 계산
- **체크**: 각 근이 원래 방정식을 만족하는지 검증

### 8.3 복소함수 계산 실수

**실수 5**: 로그 함수의 다가성 무시
- **주의점**: $$\log z$$는 무한히 많은 값을 가짐
- **해결책**: 문제에서 요구하는 값(주값 또는 일반값) 확인

**실수 6**: 복소지수 계산 시 실부/허부 분리 오류
- **체크 방법**: $$|e^z| = e^{\text{Re}(z)}$$ 확인

## 9. 고급 테크닉

### 9.1 대칭성 이용

**켤레복소수 활용**
- $$z\bar{z} = |z|^2$$
- $$(z_1 + z_2)^* = z_1^* + z_2^*$$
- $$(z_1 z_2)^* = z_1^* z_2^*$$

**실근과 허근의 분리**
$$\text{Re}(z) = \frac{z + \bar{z}}{2}$$
$$\text{Im}(z) = \frac{z - \bar{z}}{2i}$$

### 9.2 삼각함수 공식 도출

**덧셈정리**
$$e^{i(\alpha + \beta)} = e^{i\alpha}e^{i\beta}$$
$$\cos(\alpha + \beta) + i\sin(\alpha + \beta) = (\cos\alpha + i\sin\alpha)(\cos\beta + i\sin\beta)$$

실부와 허부를 비교하여:
$$\cos(\alpha + \beta) = \cos\alpha\cos\beta - \sin\alpha\sin\beta$$
$$\sin(\alpha + \beta) = \sin\alpha\cos\beta + \cos\alpha\sin\beta$$

### 9.3 급수와 적분

**지수함수 급수**
$$e^z = \sum_{n=0}^{\infty} \frac{z^n}{n!}$$

**삼각함수 급수**
$$\cos z = \sum_{n=0}^{\infty} \frac{(-1)^n z^{2n}}{(2n)!}$$
$$\sin z = \sum_{n=0}^{\infty} \frac{(-1)^n z^{2n+1}}{(2n+1)!}$$

## 10. 물리학과 공학에서의 응용

### 10.1 전기공학 응용

**교류 회로 해석**
- **전압**: $$V(t) = V_0 e^{i(\omega t + \phi)}$$
- **전류**: $$I(t) = I_0 e^{i(\omega t + \phi - \theta)}$$
- **임피던스**: $$Z = R + iX$$ ($$R$$: 저항, $$X$$: 리액턴스)

**페이저 표현**
$$\tilde{V} = V_0 e^{i\phi}$$
$$\tilde{I} = I_0 e^{i(\phi - \theta)}$$

### 10.2 신호처리 응용

**푸리에 변환**
$$F(\omega) = \int_{-\infty}^{\infty} f(t) e^{-i\omega t} dt$$

**이산 푸리에 변환**
$$X_k = \sum_{n=0}^{N-1} x_n e^{-2\pi i k n / N}$$

### 10.3 양자역학 응용

**파동함수**
$$\psi(x,t) = Ae^{i(kx - \omega t)}$$

**슈뢰딩거 방정식**
$$i\hbar \frac{\partial \psi}{\partial t} = \hat{H}\psi$$

## 11. 실전 계산 팁

### 11.1 빠른 계산법

**특수각 암기**
- $$e^{i\pi/6} = \frac{\sqrt{3}}{2} + \frac{i}{2}$$
- $$e^{i\pi/4} = \frac{1}{\sqrt{2}} + \frac{i}{\sqrt{2}}$$
- $$e^{i\pi/3} = \frac{1}{2} + \frac{i\sqrt{3}}{2}$$
- $$e^{i\pi/2} = i$$

**거듭제곱 패턴**
- $$i^1 = i$$, $$i^2 = -1$$, $$i^3 = -i$$, $$i^4 = 1$$
- $$i^n = i^{n \bmod 4}$$

### 11.2 검산 방법

**절댓값 확인**
$$|z_1 z_2| = |z_1||z_2|$$
$$|z_1/z_2| = |z_1|/|z_2|$$

**각도 확인**  
$$\arg(z_1 z_2) = \arg(z_1) + \arg(z_2) \pmod{2\pi}$$
$$\arg(z_1/z_2) = \arg(z_1) - \arg(z_2) \pmod{2\pi}$$
