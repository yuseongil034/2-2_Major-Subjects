## 🧮 핵심 적분공식 (암기 필수)

### **기본 적분공식**

$$\int x^n dx = \frac{x^{n+1}}{n+1} + C \quad (n \neq -1)$$

$$\int \frac{1}{x} dx = \ln|x| + C$$

$$\int e^x dx = e^x + C$$

$$\int a^x dx = \frac{a^x}{\ln a} + C$$

$$\int \ln x dx = x\ln x - x + C$$

### **삼각함수 적분공식**

$$\int \sin x dx = -\cos x + C$$

$$\int \cos x dx = \sin x + C$$

$$\int \tan x dx = -\ln|\cos x| + C$$

$$\int \cot x dx = \ln|\sin x| + C$$

$$\int \sec x dx = \ln|\sec x + \tan x| + C$$

$$\int \csc x dx = -\ln|\csc x + \cot x| + C$$

$$\int \sec^2 x dx = \tan x + C$$

$$\int \csc^2 x dx = -\cot x + C$$

$$\int \sec x \tan x dx = \sec x + C$$

$$\int \csc x \cot x dx = -\csc x + C$$

### **역삼각함수 적분공식**

$$\int \frac{1}{\sqrt{1-x^2}} dx = \sin^{-1} x + C$$

$$\int \frac{1}{1+x^2} dx = \tan^{-1} x + C$$

$$\int \frac{1}{x\sqrt{x^2-1}} dx = \sec^{-1} x + C$$

### **쌍곡함수 적분공식**

$$\int \sinh x dx = \cosh x + C$$

$$\int \cosh x dx = \sinh x + C$$

$$\int \tanh x dx = \ln(\cosh x) + C$$

## 🎯 적분 테크닉과 방법론

### **치환적분 (u-substitution)**

**기본 형태:** $\int f(g(x))g'(x) dx$

**Step 1:** $u = g(x)$로 치환
**Step 2:** $du = g'(x) dx$
**Step 3:** $\int f(u) du$로 변환
**Step 4:** 적분 후 $u$를 $x$로 되돌림

### **부분적분 (Integration by Parts)**

$$\int u dv = uv - \int v du$$

**LIATE 순서:** **L**og, **I**nverse trig, **A**lgebraic, **T**rig, **E**xponential

### **부분분수 분해**

$$\frac{P(x)}{Q(x)} = \sum \text{각 인수별 부분분수}$$

1. **서로 다른 1차 인수:** $\frac{A}{x-a}$
2. **중근:** $\frac{A_1}{x-a} + \frac{A_2}{(x-a)^2} + \cdots$
3. **2차 기약인수:** $\frac{Ax + B}{x^2 + bx + c}$

### **삼각치환**

1. **$\sqrt{a^2 - x^2}$:** $x = a\sin\theta$
2. **$\sqrt{a^2 + x^2}$:** $x = a\tan\theta$  
3. **$\sqrt{x^2 - a^2}$:** $x = a\sec\theta$

## 🎯 단계별 계산 예시

### **유형 1: 기본 치환적분**

$$\int (2x + 1)^5 dx$$


**Step 1:** $u = 2x + 1$로 치환



**Step 2:** $du = 2 dx$ → $dx = \frac{1}{2} du$



**Step 3:** 적분 변환

$$= \int u^5 \cdot \frac{1}{2} du = \frac{1}{2} \int u^5 du$$



**Step 4:** 계산 및 역치환

$$= \frac{1}{2} \cdot \frac{u^6}{6} + C = \frac{u^6}{12} + C = \frac{(2x + 1)^6}{12} + C$$



### **유형 2: 복잡한 치환적분**

$$\int \frac{x}{\sqrt{x^2 + 1}} dx$$

**Step 1:** $u = x^2 + 1$로 치환

**Step 2:** $du = 2x dx$ → $x dx = \frac{1}{2} du$

**Step 3:** 적분 변환

$$= \int \frac{1}{\sqrt{u}} \cdot \frac{1}{2} du = \frac{1}{2} \int u^{-1/2} du$$

**Step 4:** 계산 및 역치환

$$= \frac{1}{2} \cdot \frac{u^{1/2}}{1/2} + C = u^{1/2} + C = \sqrt{x^2 + 1} + C$$

### **유형 3: 부분적분 - 기본형**

$$\int x e^x dx$$

**Step 1:** LIATE 순서에 따라 선택
- $u = x$ (대수함수)
- $dv = e^x dx$ (지수함수)

**Step 2:** $du$와 $v$ 계산
- $du = dx$
- $v = e^x$

**Step 3:** 부분적분 공식 적용

$$= x e^x - \int e^x dx = x e^x - e^x + C = e^x(x - 1) + C$$

### **유형 4: 부분적분 - 반복형**

$$\int x^2 e^x dx$$

**Step 1:** 첫 번째 부분적분
- $u = x^2$, $dv = e^x dx$
- $du = 2x dx$, $v = e^x$

$$= x^2 e^x - \int 2x e^x dx$$

**Step 2:** 두 번째 부분적분 ($\int 2x e^x dx$)
- $u = 2x$, $dv = e^x dx$
- $du = 2 dx$, $v = e^x$

$$\int 2x e^x dx = 2x e^x - \int 2e^x dx = 2x e^x - 2e^x$$

**Step 3:** 최종 결과

$$= x^2 e^x - (2x e^x - 2e^x) + C = e^x(x^2 - 2x + 2) + C$$

### **유형 5: 부분적분 - 순환형**

$$\int e^x \sin x dx$$

**Step 1:** 첫 번째 부분적분
- $u = \sin x$, $dv = e^x dx$
- $du = \cos x dx$, $v = e^x$

$$I = e^x \sin x - \int e^x \cos x dx$$

**Step 2:** 두 번째 부분적분
- $u = \cos x$, $dv = e^x dx$
- $du = -\sin x dx$, $v = e^x$

$$\int e^x \cos x dx = e^x \cos x - \int e^x(-\sin x) dx = e^x \cos x + \int e^x \sin x dx$$

**Step 3:** 순환 관계 이용

$$I = e^x \sin x - (e^x \cos x + I)$$

$$2I = e^x \sin x - e^x \cos x$$

$$I = \frac{e^x(\sin x - \cos x)}{2} + C$$

### **유형 6: 부분분수 분해**

$$\int \frac{x + 1}{x^2 - x - 2} dx$$

**Step 1:** 분모 인수분해

$$x^2 - x - 2 = (x - 2)(x + 1)$$

**Step 2:** 부분분수 분해

$$\frac{x + 1}{(x - 2)(x + 1)} = \frac{A}{x - 2} + \frac{B}{x + 1}$$

**Step 3:** 계수 결정

$$x + 1 = A(x + 1) + B(x - 2)$$

$x = 2$: $3 = 3A$ → $A = 1$
$x = -1$: $0 = -3B$ → $B = 0$

**Step 4:** 적분 계산

$$= \int \frac{1}{x - 2} dx = \ln|x - 2| + C$$

### **유형 7: 삼각치환**

$$\int \frac{1}{\sqrt{4 - x^2}} dx$$

**Step 1:** 삼각치환 설정
- $\sqrt{4 - x^2} = \sqrt{2^2 - x^2}$ 형태
- $x = 2\sin\theta$, $dx = 2\cos\theta d\theta$

**Step 2:** 치환 후 적분

$$\sqrt{4 - x^2} = \sqrt{4 - 4\sin^2\theta} = 2\sqrt{1 - \sin^2\theta} = 2\cos\theta$$

$$= \int \frac{2\cos\theta}{2\cos\theta} d\theta = \int d\theta = \theta + C$$

**Step 3:** 역치환

$$\theta = \sin^{-1}\left(\frac{x}{2}\right)$$

$$= \sin^{-1}\left(\frac{x}{2}\right) + C$$

### **유형 8: 삼각함수의 거듭제곱**

$$\int \sin^3 x \cos^2 x dx$$

**Step 1:** 홀수 거듭제곱 처리 ($\sin^3 x = \sin^2 x \cdot \sin x$)

$$= \int \sin^2 x \cos^2 x \sin x dx$$

**Step 2:** $\sin^2 x = 1 - \cos^2 x$ 치환

$$= \int (1 - \cos^2 x) \cos^2 x \sin x dx$$

**Step 3:** $u = \cos x$, $du = -\sin x dx$로 치환

$$= -\int (1 - u^2) u^2 du = -\int (u^2 - u^4) du$$

**Step 4:** 적분 및 역치환

$$= -\left(\frac{u^3}{3} - \frac{u^5}{5}\right) + C = -\frac{\cos^3 x}{3} + \frac{\cos^5 x}{5} + C$$

### **유형 9: 로그함수 적분**

$$\int \ln x dx$$

**Step 1:** 부분적분 설정
- $u = \ln x$ (로그함수)
- $dv = dx$ (1을 곱한 것으로 간주)

**Step 2:** $du$와 $v$ 계산
- $du = \frac{1}{x} dx$
- $v = x$

**Step 3:** 부분적분 적용

$$= x \ln x - \int x \cdot \frac{1}{x} dx = x \ln x - \int dx = x \ln x - x + C$$

### **유형 10: 이상적분**

$$\int_1^{\infty} \frac{1}{x^2} dx$$

**Step 1:** 극한으로 정의

$$= \lim_{t \to \infty} \int_1^t \frac{1}{x^2} dx$$

**Step 2:** 부정적분 계산

$$= \lim_{t \to \infty} \left[-\frac{1}{x}\right]_1^t = \lim_{t \to \infty} \left(-\frac{1}{t} + 1\right)$$

**Step 3:** 극한값 계산

$$= 0 + 1 = 1$$

## 📈 정적분의 성질과 응용

### **정적분의 기본 성질**

1. **구간 분할:** $\int_a^c f(x) dx = \int_a^b f(x) dx + \int_b^c f(x) dx$

2. **우함수/기함수:**
   - 우함수: $\int_{-a}^a f(x) dx = 2\int_0^a f(x) dx$
   - 기함수: $\int_{-a}^a f(x) dx = 0$

3. **주기함수:** $\int_a^{a+T} f(x) dx = \int_0^T f(x) dx$

### **평균값 정리**

연속함수 $f(x)$의 구간 $[a,b]$에서의 평균값:

$$\bar{f} = \frac{1}{b-a} \int_a^b f(x) dx$$

### **넓이 계산**

**두 곡선 사이의 넓이:**

$$A = \int_a^b |f(x) - g(x)| dx$$

**매개변수 형태:**

$$A = \int_{\alpha}^{\beta} y \frac{dx}{dt} dt$$

### **부피 계산**

**회전체 부피 (원판법):**

$$V = \pi \int_a^b [f(x)]^2 dx$$

**회전체 부피 (껍질법):**

$$V = 2\pi \int_a^b x f(x) dx$$

### **호장의 길이**

**직교좌표:**

$$L = \int_a^b \sqrt{1 + \left(\frac{dy}{dx}\right)^2} dx$$

**매개변수:**

$$L = \int_{\alpha}^{\beta} \sqrt{\left(\frac{dx}{dt}\right)^2 + \left(\frac{dy}{dt}\right)^2} dt$$

## 🌟 급수와 수렴성

### **급수의 수렴판정법**

1. **비교판정법:** $0 \leq a_n \leq b_n$이고 $\sum b_n$ 수렴 → $\sum a_n$ 수렴

2. **비율판정법:** $\lim_{n \to \infty} \left|\frac{a_{n+1}}{a_n}\right| = L$
   - $L < 1$: 수렴
   - $L > 1$: 발산
   - $L = 1$: 판정불가

3. **근판정법:** $\lim_{n \to \infty} \sqrt[n]{|a_n|} = L$
   - $L < 1$: 수렴
   - $L > 1$: 발산

4. **적분판정법:** $f(x)$ 단조감소, 양수 → $\sum f(n)$과 $\int f(x) dx$의 수렴성 동일

### **멱급수**

$$\sum_{n=0}^{\infty} a_n (x - c)^n$$

**수렴반경:** $R = \lim_{n \to \infty} \left|\frac{a_n}{a_{n+1}}\right|$ 또는 $R = \frac{1}{\limsup_{n \to \infty} \sqrt[n]{|a_n|}}$

### **테일러 급수**

$$f(x) = \sum_{n=0}^{\infty} \frac{f^{(n)}(a)}{n!} (x - a)^n$$

**중요한 테일러 급수들:**

$$e^x = \sum_{n=0}^{\infty} \frac{x^n}{n!} = 1 + x + \frac{x^2}{2!} + \frac{x^3}{3!} + \cdots$$

$$\sin x = \sum_{n=0}^{\infty} \frac{(-1)^n x^{2n+1}}{(2n+1)!} = x - \frac{x^3}{3!} + \frac{x^5}{5!} - \cdots$$

$$\cos x = \sum_{n=0}^{\infty} \frac{(-1)^n x^{2n}}{(2n)!} = 1 - \frac{x^2}{2!} + \frac{x^4}{4!} - \cdots$$

$$\ln(1 + x) = \sum_{n=1}^{\infty} \frac{(-1)^{n+1} x^n}{n} = x - \frac{x^2}{2} + \frac{x^3}{3} - \cdots$$

$$(1 + x)^{\alpha} = \sum_{n=0}^{\infty} \binom{\alpha}{n} x^n = 1 + \alpha x + \frac{\alpha(\alpha-1)}{2!} x^2 + \cdots$$

## 🎵 푸리에 급수

### **푸리에 급수 전개**

주기 $2L$인 함수 $f(x)$:

$$f(x) = \frac{a_0}{2} + \sum_{n=1}^{\infty} \left(a_n \cos\frac{n\pi x}{L} + b_n \sin\frac{n\pi x}{L}\right)$$

**푸리에 계수:**

$$a_0 = \frac{1}{L} \int_{-L}^L f(x) dx$$

$$a_n = \frac{1}{L} \int_{-L}^L f(x) \cos\frac{n\pi x}{L} dx$$

$$b_n = \frac{1}{L} \int_{-L}^L f(x) \sin\frac{n\pi x}{L} dx$$

### **특수한 경우들**

**우함수:** $b_n = 0$ (코사인 급수만)

**기함수:** $a_n = 0$ (사인 급수만)

**주기 $2\pi$:** $L = \pi$

$$a_n = \frac{1}{\pi} \int_{-\pi}^{\pi} f(x) \cos(nx) dx$$

$$b_n = \frac{1}{\pi} \int_{-\pi}^{\pi} f(x) \sin(nx) dx$$

### **복소 푸리에 급수**

$$f(x) = \sum_{n=-\infty}^{\infty} c_n e^{in\pi x/L}$$

$$c_n = \frac{1}{2L} \int_{-L}^L f(x) e^{-in\pi x/L} dx$$

## ⚠️ 실수하기 쉬운 포인트

### **주의점 1: 치환적분에서 적분 구간**
정적분에서 치환할 때는 적분구간도 함께 변환해야 함

### **주의점 2: 부분적분에서 LIATE 순서**
- **L**ogarithmic (로그)
- **I**nverse trigonometric (역삼각함수)
- **A**lgebraic (대수함수)
- **T**rigonometric (삼각함수)
- **E**xponential (지수함수)

### **주의점 3: 이상적분 수렴성 확인**
특이점에서의 거동과 무한대에서의 거동을 모두 확인

### **주의점 4: 절댓값 처리**
- $\int \frac{1}{x} dx = \ln|x| + C$ (절댓값 필수)
- 정적분에서 피적분함수가 부호를 바꾸는 경우 구간 분할 필요

### **주의점 5: 삼각치환 후 역치환**
삼각치환 후 반드시 원래 변수로 되돌려야 함

## 🚀 실전 풀이 팁

### **Tip 1: 적분 방법 선택 순서**
1. 기본 공식으로 바로 해결 가능한지 확인
2. 간단한 치환으로 해결 가능한지 확인
3. 부분적분 적용 (LIATE 순서)
4. 부분분수나 삼각치환 고려
5. 복잡한 경우 여러 방법 조합

### **Tip 2: 계산 실수 줄이기**
- 치환 후 모든 변수를 새로운 변수로 완전히 바꾸기
- 부분적분에서 $dv$ 선택할 때 적분하기 쉬운 것 선택
- 중간 계산 단계마다 검산

### **Tip 3: 대칭성 활용**
- 우함수/기함수의 성질 적극 활용
- 주기함수의 성질 이용
- $\int_0^a f(x) dx = \int_0^a f(a-x) dx$ 활용

### **Tip 4: 복잡한 적분의 변형**
- $\int \frac{1}{\sqrt{ax^2 + bx + c}} dx$ → 완전제곱식으로 변형
- $\int R(\sin x, \cos x) dx$ → 바일러슈트라스 치환 $t = \tan(x/2)$ 고려
- 분모의 차수가 높으면 부분분수 우선 고려

### **Tip 5: 수치적 검증**
- 간단한 값 대입해서 결과 확인
- 부정적분의 도함수가 원함수와 같은지 확인
- 정적분의 경우 기하학적 의미로 대략적 크기 추정

### **Tip 6: 급수 수렴성 빠른 판단**
- $\frac{1}{n^p}$ 형태: $p > 1$이면 수렴
- 기하급수: $|r| < 1$이면 수렴
- 교대급수: 단조감소하고 극한이 0이면 수렴

### **Tip 7: 테일러 급수 활용**
- $x \to 0$ 근사에서 적극 활용
- 함수의 극한값 계산에 유용
- 근사 계산에서 필요한 항까지만 계산

### **Tip 8: 푸리에 급수 계산**
- 대칭성으로 계수 절반 줄이기
- 삼각함수 직교성 적극 활용
- 단위 구간으로 정규화해서 계산 후 변환
