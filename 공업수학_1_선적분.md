# 📏 선적분(매개변수,양함수) 실전 가이드

## 🔧 1. 기본 공식

### 📐 매개변수 곡선에서의 선적분
곡선 $$C: \vec{r}(t) = (x(t), y(t)), \quad a \leq t \leq b$$

**제1종 선적분 (호장에 대한 적분) 📊**
$$\int_C f(x,y) \, ds = \int_a^b f(x(t), y(t)) \sqrt{\left(\frac{dx}{dt}\right)^2 + \left(\frac{dy}{dt}\right)^2} \, dt$$

**제2종 선적분 (좌표에 대한 적분) ⚡**
$$\int_C P \, dx + Q \, dy = \int_a^b \left[ P(x(t), y(t)) \frac{dx}{dt} + Q(x(t), y(t)) \frac{dy}{dt} \right] dt$$

### 📈 양함수에서의 선적분
곡선 $$C: y = f(x), \quad a \leq x \leq b$$

**제1종 선적분 📊**
$$\int_C g(x,y) \, ds = \int_a^b g(x, f(x)) \sqrt{1 + \left(\frac{dy}{dx}\right)^2} \, dx$$

**제2종 선적분 ⚡**
$$\int_C P \, dx + Q \, dy = \int_a^b \left[ P(x, f(x)) + Q(x, f(x)) \frac{dy}{dx} \right] dx$$

## 🎯 2. 자주 나오는 문제 유형

### 📏 유형 1: 호장 길이 구하기
- $$\int_C 1 \, ds$$ 형태
- $$ds = \sqrt{\left(\frac{dx}{dt}\right)^2 + \left(\frac{dy}{dt}\right)^2} \, dt$$

### ⚖️ 유형 2: 질량/무게중심 문제
- 밀도함수 $$\rho(x,y)$$가 주어진 경우
- 질량: $$M = \int_C \rho(x,y) \, ds$$
- 무게중심: $$\bar{x} = \frac{1}{M}\int_C x\rho(x,y) \, ds$$

### 🔄 유형 3: 벡터장에서의 일 계산
- $$\vec{F} = (P, Q)$$인 벡터장에서의 일
- $$W = \int_C \vec{F} \cdot d\vec{r} = \int_C P \, dx + Q \, dy$$

### 🔀 유형 4: Green 정리 활용 문제
- 단순폐곡선에서 $$\oint_C P \, dx + Q \, dy = \iint_D \left(\frac{\partial Q}{\partial x} - \frac{\partial P}{\partial y}\right) dA$$

## 🛠️ 3. 계산 테크닉

### 💡 핵심 계산 스킬

**1️⃣ 매개변수 미분 빠르게 계산하기**
- $$\frac{d}{dt}[t^n] = nt^{n-1}$$
- $$\frac{d}{dt}[\sin t] = \cos t$$, $$\frac{d}{dt}[\cos t] = -\sin t$$
- $$\frac{d}{dt}[e^t] = e^t$$

**2️⃣ 제곱근 계산 최적화**
- $$\sqrt{a^2 + b^2}$$ 형태는 삼각함수 치환 고려
- $$\sqrt{1 + \tan^2 \theta} = \sec \theta$$
- $$\sqrt{1 + \sin^2 t}$$ 같은 형태는 수치적으로 계산

**3️⃣ 대칭성 활용**
- 함수가 홀함수이고 적분구간이 대칭이면 $$\int_{-a}^a f(x) dx = 0$$
- 함수가 짝함수이면 $$\int_{-a}^a f(x) dx = 2\int_0^a f(x) dx$$

## 📝 4. 단계별 계산 예시

### 🔵 예시 1: 매개변수 곡선 호장 길이
**문제**: $$x = t^2, y = t^3, 0 \leq t \leq 1$$인 곡선의 길이

**풀이**:
1. $$\frac{dx}{dt} = 2t$$, $$\frac{dy}{dt} = 3t^2$$

2. $$ds = \sqrt{(2t)^2 + (3t^2)^2} dt = \sqrt{4t^2 + 9t^4} dt = t\sqrt{4 + 9t^2} dt$$

3. $$L = \int_0^1 t\sqrt{4 + 9t^2} \, dt$$

4. 치환: $$u = 4 + 9t^2$$, $$du = 18t \, dt$$
   
5. $$L = \frac{1}{18}\int_4^{13} \sqrt{u} \, du = \frac{1}{27}[u^{3/2}]_4^{13} = \frac{13^{3/2} - 8}{27}$$

### 🟢 예시 2: 양함수에서 제2종 선적분
**문제**: $$\int_C xy \, dx + x^2 \, dy$$, 여기서 $$C: y = x^2, 0 \leq x \leq 1$$

**풀이**:
1. $$y = x^2$$이므로 $$\frac{dy}{dx} = 2x$$

2. $$\int_C xy \, dx + x^2 \, dy = \int_0^1 [x \cdot x^2 + x^2 \cdot 2x] dx$$

3. $$= \int_0^1 [x^3 + 2x^3] dx = \int_0^1 3x^3 dx$$

4. $$= \left[\frac{3x^4}{4}\right]_0^1 = \frac{3}{4}$$

### 🟡 예시 3: 원형 매개변수에서 제1종 선적분
**문제**: $$\int_C (x^2 + y^2) ds$$, 여기서 $$C: x = \cos t, y = \sin t, 0 \leq t \leq 2\pi$$

**풀이**:
1. $$\frac{dx}{dt} = -\sin t$$, $$\frac{dy}{dt} = \cos t$$

2. $$ds = \sqrt{\sin^2 t + \cos^2 t} dt = dt$$

3. $$x^2 + y^2 = \cos^2 t + \sin^2 t = 1$$

4. $$\int_C (x^2 + y^2) ds = \int_0^{2\pi} 1 \cdot dt = 2\pi$$

### 🔴 예시 4: Green 정리 활용
**문제**: $$\oint_C (x^2 - y) dx + (x + y^2) dy$$, 여기서 $$C$$는 $$x^2 + y^2 = 1$$의 반시계방향

**풀이**:
1. $$P = x^2 - y$$, $$Q = x + y^2$$

2. $$\frac{\partial Q}{\partial x} = 1$$, $$\frac{\partial P}{\partial y} = -1$$

3. Green 정리에 의해:
   $$\oint_C P dx + Q dy = \iint_D (1 - (-1)) dA = 2\iint_D dA$$

4. 단위원의 넓이는 $$\pi$$이므로: $$2 \cdot \pi = 2\pi$$

## ⚠️ 5. 실수하기 쉬운 포인트

### 🚨 주의사항

**1️⃣ 매개변수 방향성**
- 곡선의 방향이 바뀌면 제2종 선적분의 부호가 바뀜
- $$\int_{C^-} P dx + Q dy = -\int_C P dx + Q dy$$

**2️⃣ 제곱근 계산 실수**
- $$\sqrt{(\frac{dx}{dt})^2 + (\frac{dy}{dt})^2}$$에서 제곱 빼먹지 않기
- 음수 구간에서 절댓값 고려: $$\sqrt{t^2} = |t|$$

**3️⃣ 적분 구간 설정**
- 매개변수의 시작점과 끝점 정확히 확인
- 폐곡선인지 열린곡선인지 구분

**4️⃣ Green 정리 적용 조건**
- 단순폐곡선이어야 함
- 영역 $$D$$가 단순연결영역이어야 함
- 반시계방향으로 적분해야 함

## 🎯 6. 실전 풀이 팁

### 💪 효율적인 계산 전략

**🔍 Step 1: 문제 타입 빠르게 판단**
- 제1종($$ds$$)인지 제2종($$dx, dy$$)인지 확인
- 매개변수인지 양함수인지 확인

**🔍 Step 2: 적절한 공식 선택**
- Green 정리 적용 가능한지 먼저 체크
- 대칭성 활용 가능한지 확인

**🔍 Step 3: 계산 단순화**
- 치환적분 활용
- 삼각함수 항등식 활용
- 부분적분 고려

### 🏆 고득점 전략

**✅ 계산 검증 방법**
- 차원 분석으로 답의 합리성 확인
- 특수한 경우(원점, 축 위의 점)로 검산
- 물리적 의미 고려 (길이는 양수, 일의 방향성 등)

**✅ 시간 단축 팁**
- 자주 쓰이는 치환 패턴 암기
- 표준형 적분 결과 외우기
- 계산기 활용법 숙지

**✅ 실수 방지법**
- 매개변수 미분 계산 후 바로 검산
- 적분 구간과 함수 정의역 일치 확인
- 최종 답에서 단위 확인
